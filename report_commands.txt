# Разница между git log и git show
## git log
## git show
- Можно указать конкретный коммит: git show <hash>
1. git log показывает список коммитов, git show показывает один коммит детально
2. git log показывает только метаданные, git show показывает изменения в коде
3. git log используется для навигации по истории, git show для анализа изменений
